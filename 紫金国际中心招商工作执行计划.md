# 紫金国际中心商业裙楼 - 招商工作执行计划

## 一、招商工作总体目标

- **总体招商率**：开业前达到95%以上
- **品牌质量**：确保30%以上为国际国内一线品牌
- **业态组合**：实现科学合理的业态配置，满足不同消费需求
- **开业时间**：确保2025年按计划顺利开业

## 二、招商团队组建与分工

### 招商团队架构

- **招商总监**：1名，负责整体招商策略制定和团队管理
- **招商经理**：3名，分别负责一层、二层、三层的招商工作
- **招商专员**：6名，协助招商经理开展具体招商工作
- **市场推广专员**：2名，负责招商推广和活动策划
- **客户关系专员**：1名，负责客户跟进和关系维护
- **法务专员**：1名，负责合同审核和法律事务

### 团队职责分工

#### 招商总监职责
- 制定整体招商策略和计划
- 审核招商政策和合作条件
- 负责重点品牌的高层对接
- 统筹协调各部门工作
- 定期向公司领导汇报招商进展

#### 招商经理职责
- 制定本楼层招商计划和目标
- 负责目标品牌的筛选和对接
- 谈判和确定合作条件
- 管理和指导招商专员工作
- 定期向招商总监汇报工作进展

#### 招商专员职责
- 收集和整理目标品牌信息
- 进行品牌初步接触和沟通
- 带看现场和介绍项目
- 准备招商资料和提案
- 跟进客户意向和签约进度

## 三、招商工作时间节点

### 第一阶段：前期准备（1-2个月）

#### 第1个月
- **团队组建**：招聘和组建招商团队
- **培训**：对招商团队进行项目知识、招商技巧等培训
- **市场调研**：对三亚市商业市场进行全面调研
- **资料准备**：制作招商手册、PPT、效果图等资料
- **招商政策制定**：确定租金标准、优惠政策等

#### 第2个月
- **目标品牌筛选**：确定各楼层重点招商目标品牌
- **品牌资源整理**：建立品牌数据库和联系方式
- **招商中心建设**：完成招商中心的装修和布置
- **媒体资源对接**：联系媒体，准备招商宣传
- **招商启动会**：召开项目招商启动会议

### 第二阶段：主力品牌招商（3-6个月）

#### 第3-4个月
- **一线品牌对接**：开始对接国际一线品牌和主力店
- **高层拜访**：招商总监带队拜访重点品牌总部
- **招商推介会**：举办第一场大型招商推介会
- **媒体宣传**：开始项目招商宣传推广
- **品牌考察接待**：接待品牌方前来项目考察

#### 第5-6个月
- **主力店签约**：完成3-5个核心主力店的签约
- **品牌组合优化**：根据主力店情况调整品牌组合
- **第二场推介会**：针对特色品牌举办专场推介会
- **招商进度跟踪**：定期跟踪和汇报招商进度
- **政策调整**：根据招商情况适时调整招商政策

### 第三阶段：全面招商（7-12个月）

#### 第7-9个月
- **全面铺开招商**：扩大招商范围，全面推进各业态招商
- **品牌补充**：根据已签约品牌情况，补充相应业态品牌
- **招商活动**：举办各类小型招商活动和品牌对接会
- **签约加速**：加快签约速度，提高招商率
- **客户关系维护**：加强与已签约品牌的沟通和服务

#### 第10-12个月
- **招商收尾**：重点攻克剩余铺位，确保招商率达标
- **品牌调整**：对已签约品牌进行优化和调整
- **开业准备**：协助已签约品牌进行开业准备
- **招商总结**：对招商工作进行全面总结
- **运营交接**：与运营团队进行工作交接

### 第四阶段：开业筹备（开业前3个月）

- **装修跟进**：监督和协助品牌方进行店铺装修
- **开业活动策划**：策划开业活动和营销推广方案
- **人员培训**：协助品牌方进行员工培训
- **开业宣传**：加大开业宣传力度
- **开业仪式准备**：筹备开业仪式和相关活动

## 四、招商推广策略

### 线上推广

1. **官方网站**：建立项目官方网站，展示项目信息和招商动态
2. **社交媒体**：利用微信公众号、微博、抖音等平台进行宣传
3. **行业网站**：在商业地产、品牌招商等专业网站发布信息
4. **搜索引擎优化**：优化项目关键词，提高搜索可见度
5. **线上招商会**：举办线上招商推介会，扩大影响力

### 线下推广

1. **招商中心**：打造高品质招商中心，展示项目形象
2. **户外广告**：在项目周边和城市主要位置设置户外广告牌
3. **行业展会**：参加国内外商业地产和品牌招商展会
4. **推介会**：举办专场招商推介会和品牌对接会
5. **圈层活动**：举办高端圈层活动，接触潜在品牌商家

### 媒体合作

1. **报纸杂志**：在商业地产和时尚杂志发布项目信息
2. **电视广播**：在当地电视台和广播电台进行宣传
3. **自媒体合作**：与商业地产和时尚领域的自媒体合作
4. **新闻发布会**：举办项目新闻发布会，提高知名度

## 五、招商工作流程

### 品牌接触流程

1. **目标品牌确定**：根据招商计划确定目标品牌
2. **信息收集**：收集品牌详细信息和联系方式
3. **初次接触**：通过电话、邮件等方式进行初步沟通
4. **资料发送**：发送项目招商资料给意向品牌
5. **意向确认**：确认品牌意向，安排进一步接触

### 品牌谈判流程

1. **需求沟通**：了解品牌具体需求和期望
2. **铺位推荐**：根据品牌需求推荐合适铺位
3. **条件洽谈**：洽谈租金、租期、优惠政策等条件
4. **方案制定**：制定个性化合作方案
5. **高层会晤**：安排双方高层会面，推进合作

### 合同签订流程

1. **合同起草**：根据谈判结果起草合同文本
2. **法务审核**：法务专员审核合同条款
3. **合同确认**：与品牌方确认合同细节
4. **合同签订**：正式签订租赁合同
5. **费用收取**：收取租赁押金和首期租金

### 后续服务流程

1. **交接手续**：办理铺位交接手续
2. **装修指导**：提供装修设计和施工指导
3. **开业协助**：协助品牌方进行开业准备
4. **运营支持**：提供运营管理和营销支持
5. **定期回访**：定期回访，了解经营情况

## 六、招商工作考核与激励

### 考核指标

1. **招商率**：各楼层铺位招商完成率
2. **品牌质量**：一线品牌和优质品牌占比
3. **签约速度**：每月新增签约面积和品牌数量
4. **客户满意度**：客户对招商服务的满意度评价
5. **团队协作**：团队成员之间的协作情况

### 激励机制

1. **佣金奖励**：根据招商成果发放佣金奖励
2. **绩效奖金**：根据考核指标完成情况发放绩效奖金
3. **晋升机会**：表现优秀的团队成员给予晋升机会
4. **培训机会**：提供专业培训和学习机会
5. **团队活动**：定期组织团队活动，增强团队凝聚力

## 七、招商风险防控

### 风险识别

1. **市场风险**：市场环境变化导致招商困难
2. **品牌风险**：品牌方经营不善或中途退出
3. **政策风险**：政策变化影响商业经营
4. **竞争风险**：周边商业项目竞争激烈
5. **合作风险**：合同执行过程中出现纠纷

### 风险应对措施

1. **市场风险应对**：
   - 加强市场监测，及时调整招商策略
   - 多元化品牌选择，降低单一依赖

2. **品牌风险应对**：
   - 严格品牌筛选，选择有实力和信誉的品牌
   - 建立品牌经营监测机制，及时发现问题

3. **政策风险应对**：
   - 密切关注政策动向，提前做好应对准备
   - 与政府部门保持良好沟通

4. **竞争风险应对**：
   - 突出项目特色和优势，差异化竞争
   - 提供更具吸引力的招商政策

5. **合作风险应对**：
   - 完善合同条款，明确双方权利义务
   - 建立健全客户关系管理体系

## 八、招商工作支持保障

### 资源支持

1. **资金支持**：确保招商工作所需的资金投入
2. **人力支持**：合理配置人力资源，确保招商团队稳定
3. **物料支持**：及时提供所需的招商物料和宣传资料
4. **技术支持**：提供必要的技术支持和信息系统

### 后勤保障

1. **办公环境**：提供舒适的办公环境和必要的办公设备
2. **交通住宿**：为外出招商和客户接待提供交通和住宿支持
3. **餐饮服务**：提供工作餐和客户接待餐饮服务
4. **安全保障**：确保办公场所和招商活动的安全

## 九、招商工作总结与评估

### 阶段性总结

1. **月度总结**：每月召开招商工作会议，总结当月工作，规划下月任务
2. **季度总结**：每季度进行一次全面总结，评估招商进度和效果
3. **半年总结**：半年进行一次深入分析，调整招商策略和计划

### 最终评估

1. **招商成果评估**：评估总体招商率、品牌质量、业态组合等指标
2. **团队表现评估**：评估招商团队的整体表现和个人贡献
3. **经验教训总结**：总结招商过程中的经验和教训
4. **报告编写**：编写招商工作总结报告，为后续项目提供参考
